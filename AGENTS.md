# AGENTS.md

## 프로젝트 기준 문서

이 문서는 Unity 2D Dungeon Survivor 프로젝트를 AI와 함께 개발할 때 반드시 지켜야 하는 기준 문서다.

AI는 이 문서를 최우선으로 따르며, 임의로 구조를 새로 만들거나 기존 시스템을 중복 생성하지 않는다.

---

## 1. 프로젝트 개요

- 프로젝트명: 2D Dungeon Survivor
- 장르: 2D 가로형 생존 로그라이크 / 뱀서류
- 개발 엔진: Unity
- 현재 Unity 버전: 6000.3.10f1
- 현재 메인 씬: `Assets/_Project/Scenes/Main.unity`
- 개발 기준: 정식 출시 가능 구조를 지향
- 현재 목표: 5분 MVP 안정화 후 10분 수직 슬라이스로 확장
- 그래픽 방향: 도트 기반 다크 판타지, 스톤샤드풍 참고
- 개발 방식: 한 번에 하나의 기능만 추가하고, 기존 구조를 보존한다.

---
